# Digital FTE Dashboard Skills

Skills for managing and updating the AI Employee dashboard.

## Dashboard Location

```
ai_employee_vault/Dashboard.md
```

## Skills

### Read Dashboard

Read the current dashboard state.

```bash
/read_file path="ai_employee_vault/Dashboard.md"
```

### Update Dashboard Stats

Update the statistics on the dashboard.

```python
# Count files in each folder
needs_action = len(list(Path("ai_employee_vault/Needs_Action").glob("*.md")))
plans = len(list(Path("ai_employee_vault/Plans").glob("*.md")))
pending = len(list(Path("ai_employee_vault/Pending_Approval").glob("*.md")))
done_today = count_done_today()

# Update Dashboard.md with new counts
```

### Example: Generate Weekly Briefing

```bash
# Run on Sunday evening

# 2. Count completed tasks from Done/ folder

# 4. Identify bottlenecks from task timestamps

```

## Best Practices

1. **Update Frequency:** Update dashboard after each task cycle
2. **Timestamp:** Always update "Last full refresh" timestamp
3. **Accuracy:** Ensure counts match actual folder contents
4. **Briefings:** Save weekly briefings in Briefings/ folder
5. **Archive:** Move old briefings to archive after quarter ends
